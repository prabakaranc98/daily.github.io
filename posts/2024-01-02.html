Mathematical Beauty with MathJax

One of the things I wanted for this journal was the ability to write mathematical equations. Using MathJax, I can now write equations inline or as display blocks.

Inline Math

For example, I can write about the famous equation \(E = mc^2\) right in the middle of a sentence. Or mention that the quadratic formula is \(x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}\).

Display Math

For more complex equations, I can use display mode. Here's the beautiful Euler's identity:

$$e^{i\pi} + 1 = 0$$

And here's the Gaussian (normal) distribution:

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}$$

Code Snippets

I can also share code snippets with proper formatting. Here's a simple Python function:

def fibonacci(n):
    """Calculate the nth Fibonacci number."""
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Print first 10 Fibonacci numbers
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")

Fun Math Fact

Did you know? The sum of all positive integers approaches negative one-twelfth:

$$1 + 2 + 3 + 4 + \cdots = -\frac{1}{12}$$

This surprising result (from Ramanujan summation) has applications in string theory and quantum physics!
